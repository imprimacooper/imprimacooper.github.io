# Banners e Faixas — Arquivos Estáticos

Esta pasta contém a landing page **Banners e Faixas Personalizados** da Imprima Cooper pronta para hospedagem estática.

## Conteúdo

- `index.html` — página completa (HTML + CSS inline do React, sem hydration)
- `favicon.ico` — ícone do site
- `assets/` — imagens e folha de estilos
  - `banner-hero-CHCu5kIj.jpg`
  - `banner-madeira-D7ZpFCVv.jpg`
  - `banner-ilhos-C2pLwhEU.jpg`
  - `banner-production-WiEA3E6W.jpg`
  - `imprima-cooper-logo.png`
  - `styles-CoLmX-IK.css`

## Como hospedar

1. Faça upload de **todos** os arquivos e pastas para seu servidor/hospedagem.
2. Mantenha a estrutura de pastas (`index.html` na raiz e `assets/` ao lado).
3. Acesse `index.html` no navegador.

## Hospedagens compatíveis

- Netlify (drag & drop da pasta)
- Vercel (importar pasta estática)
- Cloudflare Pages
- GitHub Pages
- Apache/nginx/Caddy
- Qualquer hospedagem compartilhada com suporte a HTML estático

## Observações

- A página é 100% estática: não requer Node.js, React, banco de dados ou servidor SSR.
- Os links de navegação apontam para https://www.imprimacooper.com.br/.
- As fontes (Montserrat e Inter) são carregadas do Google Fonts.
