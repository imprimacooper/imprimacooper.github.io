Imprima Cooper — Landing Page de Adesivos (Arquivos Estáticos)
===============================================================

Esta pasta contém a versão estática da landing page de Adesivos Personalizados,
pronta para hospedar em qualquer servidor de hospedagem compartilhada (cPanel,
FTP, Cloudflare Pages, Netlify, Vercel, etc.).

Conteúdo:
- index.html          (página principal)
- styles.css          (estilos otimizados)
- adesivo-hero.jpg    (imagem do hero)
- adesivo-application.jpg
- adesivo-vehicle.jpg
- adesivo-rolls.jpg
- imprima-cooper-logo.png
- favicon.ico

Como publicar:
1. Envie todos os arquivos desta pasta para a raiz do seu domínio ou subpasta.
2. Se publicar na raiz, a página estará em: https://seudominio.com/
3. Se publicar em uma subpasta, ex: /adesivos/ , acesse: https://seudominio.com/adesivos/

Observação: não é necessário Node.js ou servidor React no destino.
A página é 100% estática (HTML + CSS + imagens).
