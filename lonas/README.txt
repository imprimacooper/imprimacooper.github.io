=== Imprima Cooper — Landing Page Estática (Lonas) ===

Arquivos nesta pasta:
- index.html          -> Página principal (abra no navegador ou suba no servidor)
- styles.css          -> Estilos processados (Tailwind + animações)
- imprima-cooper-logo.png -> Logo
- lona-hero.jpg       -> Imagem do hero
- lona-print.jpg      -> Imagem sobre impressão
- lona-rolls.jpg      -> Imagem de lonas
- lona-event.jpg      -> Imagem de eventos
- favicon.ico         -> Ícone da aba

Como hospedar:
1. Envie TODOS os arquivos desta pasta para a raiz do seu servidor web (cPanel, FTP, Vercel, Netlify, Cloudflare Pages, etc.).
2. Certifique-se de que index.html esteja na raiz.
3. Acesse pelo navegador.

Observações:
- A página é 100% estática (HTML + CSS + imagens). Não precisa de Node.js ou servidor React.
- Os botões apontam para os links configurados (login, categoria de lonas, WhatsApp, site principal).
- Para atualizar textos/cores, edite o index.html (conteúdo) ou styles.css (estilos).
- Para trocar imagens, substitua os arquivos .jpg/.png mantendo os mesmos nomes.
